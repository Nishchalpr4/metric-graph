## 🟦 THE 11 BASE METRICS (Measure from Business)

These 11 are what you import via CSV or Sync:

### Financial Layer (6 metrics)
```
┌─────────────────────────────────────┐
│ 💰 FINANCIAL METRICS                │
├─────────────────────────────────────┤
│ commission_rate    %  ← Take % from orders
│ delivery_charges   ₹B ← Revenue from delivery
│ discounts          ₹B ← Discounts given
│ marketing_spend    ₹B ← Ad spend
│ aov                ₹  ← Price per order
│ orders             M  ← Number of orders
└─────────────────────────────────────┘
```

### User Layer (2 metrics)
```
┌─────────────────────────────────────┐
│ 👥 USER METRICS                     │
├─────────────────────────────────────┤
│ active_users       M  ← Monthly active
│ new_users          M  ← Monthly acquired
└─────────────────────────────────────┘
```

### Operations Layer (3 metrics)
```
┌─────────────────────────────────────┐
│ ⚙️  OPERATIONAL METRICS             │
├─────────────────────────────────────┤
│ basket_size        items ← Items per order
│ restaurant_partners K    ← Number of sellers
│ pricing_index      idx  ← Price level
└─────────────────────────────────────┘
```

**Total Base: 11 metrics (all use `is_base: true`)**
